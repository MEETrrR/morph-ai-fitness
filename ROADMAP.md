### 四、 关键战役避坑指南 (红线警告)
1. **关于前端框架**：严禁零基础直接上手 Next.js App Router。必须先用 Vite+React 搞懂客户端渲染和组件状态。
2. **关于AI框架**：严禁第一个产品直接使用 LangChain。必须先用 Python 原生 HTTP 库裸调大模型 API，理解底层协议后再用框架。
3. **关于代码生成**：AI是搬砖工，你是架构师。**核心代码必须逐行Review**，严禁把API Key硬编码在前端。
4. **关于找实习**：民办本科**不要海投大厂官网**。必须通过 GitHub、技术社区、线下大会走"主管直推"内推路线。
